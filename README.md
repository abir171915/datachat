# DataChat

An agentic data analyst that explores CSVs and answers natural-language questions using Claude and LangGraph.

**Status:** Early development — building in the open.

## The idea

Upload a CSV → ask questions in plain English → an AI agent autonomously explores the data, runs stats, generates charts, and summarises findings.

## Stack (evolving)

- **LLM:** Claude (Anthropic API)
- **Agent framework:** LangGraph
- **Backend / frontend:** FastAPI + Streamlit
- **Observability:** Langfuse
- **Deployment:** Docker (AWS target)
